How Does it Work?


Inputting the Reach ID

Users start by entering a specific Reach ID into the input field provided.


API Request to NWM

Once the Reach ID is entered, the application sends a request to the NOAA NWPS API to fetch the forecast data for the selected Reach ID and forecast type.


Processing the API Response

The application checks the API response for any errors and then parses the JSON data received.


Data Extraction

The relevant data, including timestamps and streamflow values, is extracted from the JSON response.


Updating the Table

The extracted data is then used to update the table, displaying the forecast information in a clear and organized manner.


Chart Visualization

Using Chart.js, the application creates or updates a chart to visually represent the forecast data.


Error Management

If any errors occur during the API call or data processing, the application handles them gracefully and informs the user with appropriate messages.
